  🎯 简化版功能清单

  ✨ Phase 1：基础聊天（第 1-2 周）

  核心功能：
  - 用户发送纯文本消息
  - AI 流式返回回复
  - 消息列表显示（用户+AI）
  - 基础 Markdown 渲染（粗体、斜体、列表、代码块）
  - 响应式布局（移动端+桌面端）

  技术栈：
  - Next.js 15 App Router
  - Vercel AI SDK (useChat)
  - Tailwind CSS
  - shadcn/ui 基础组件

  代码量： ~500-800 行

  ---
  ✨ Phase 2：聊天历史（第 3 周）

  核心功能：
  - 聊天会话保存到数据库
  - 侧边栏显示历史会话列表
  - 点击切换会话
  - 删除会话
  - 新建会话

  技术栈：
  - Drizzle ORM
  - PostgreSQL / SQLite（本地开发）
  - SWR（数据获取）

  数据库设计（简化版）：
  -- 只需要 2 个表
  chats (id, userId, title, createdAt)
  messages (id, chatId, role, content, createdAt)

  代码量： +300 行

  核心功能：
  - 邮箱+密码登录/注册
  - 会话管理（每个用户看自己的聊天）
  - 退出登录

  技术栈：
  - NextAuth.js（简化配置）
  - 或 Clerk（更简单）

  代码量： +200 行

  export function ChatInput({ onSend }: { onSend: (text: string) => void }) 
  {
    const [input, setInput] = useState('')

    return (
      <form onSubmit={(e) => {
        e.preventDefault()
        onSend(input)
        setInput('')
      }}>
        <textarea
          value={input}
          onChange={(e) => setInput(e.target.value)}
          placeholder="Send a message..."
          className="w-full rounded-lg border p-3"
        />
        <button type="submit" className="mt-2 rounded-lg bg-blue-500 px-4 
  py-2 text-white">
          发送
        </button>
      </form>
    )
  }
