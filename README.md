# 🐘 Elly Chat

**Elly Chat** is an AI-powered chat platform — elegant, fast, and designed for deep thinking.  
Inspired by the elephant’s calm **memory** and **wisdom**, Elly Chat helps you chat, learn, and create with focus.

> 🧠 “Think deeply. Chat simply.”

---

## ✨ Features

- 🤖 Multi-model AI chat (OpenAI, Anthropic, Gemini, local models, etc.)
- ⚙️ Built with **Next.js 16**, **React**, and **Tailwind v4**
- 🎨 Clean and minimal UI, inspired by modern productivity tools
- 🔐 Focus on privacy — no tracking, no ads
- 🚀 Fast edge deployment (Cloudflare / Vercel)

---

## 🧩 Tech Stack

| Layer      | Technology                   |
| ---------- | ---------------------------- |
| Frontend   | Next.js 16 (App Router)      |
| UI         | Tailwind CSS v4 + shadcn/ui  |
| Backend    | Edge functions / AI Gateway  |
| State      | React hooks + server actions |
| Deployment | Vercel / Cloudflare Workers  |
| Language   | TypeScript + JSX             |
